# Introduction 🌀

Welcome to **DocFlow**, a minimal, blazing-fast, static-site documentation engine designed for developers. It converts directories of raw Markdown and MDX files into premium, responsive developer hubs with client-side search.

> [!NOTE]
> This site was built automatically by the DocFlow CLI. You can edit this file at `docs/index.md` and refresh the browser!

## ⚡ Key Features

* **Instant Builds:** Under 10ms compiling times for standard developer project wikis.
* **Fuzzy Local Search:** Search matches titles, descriptions, and page bodies on-the-fly.
* **Glow & Glassmorphism Aesthetics:** Premium dark-mode layout optimized for code readability.
* **GitHub-Style Alerts:** Support for native `[!NOTE]`, `[!TIP]`, and `[!WARNING]` blockquotes.

## 🛠️ Quick Steps
1. Add more `.md` files inside the `docs/` directory.
2. Add frontmatter tags (like `title` and `order`) at the top of files.
3. Keep the dev server running to preview edits instantly on save!
